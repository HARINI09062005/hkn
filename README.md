# Smart Budget Scheduler

A comprehensive budget management tool for student chapters.

## Frontend Setup

The frontend is built with React, TypeScript, Vite, and Tailwind CSS.

### Prerequisites
- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. Navigate to the client directory:
   ```bash
   cd client
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser at `http://localhost:5173`.

## Features

- **Dashboard**: Real-time financial overview.
- **Transactions**: Track expenses and incomes with detailed status steps.
- **Deadlines**: Manage financial milestones.
- **Reports**: Export data for external use.
